# Grove-NeoPixel-Multiboard

Arduino Grove NeoPixel Multiboard (4x NeoPixel)

Code for handling 4 different digital Pins for NeoPixel Control

## Grove Hardware

Used Hardware: 

* Arduino Uno // Grove **Lotus**
* NeoPixel 4 Types
  * Ring
  * Bar
  * Stripe
  * Button (with one NeoPixel)
* PowerSupply
* LCD Screen
* LED Button
